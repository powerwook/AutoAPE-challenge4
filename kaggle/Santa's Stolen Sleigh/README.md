# 사용한 방법 & 알고리즘
 *haversine 모듈: 두 위도와 경도 사이의 거리를 계산하는 데 사용됩니다.
 *bb_sort 함수에서는 순열 및 조합 알고리즘을 사용하여 최적의 솔루션을 찾습니다.
 *path_opt_test 함수에서 haversine 공식을 사용하여 두 지점 사이의 거리를 계산하고 경로 최적화 알고리즘을 사용하여 최단 경로를 찾아 총 비용을 줄입니다.
